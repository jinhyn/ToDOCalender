# ToDOCalender 🗓️✅

**할 일 관리 + 캘린더 기능**을 제공하는 프로젝트입니다.  
프론트엔드(React)와 백엔드(Node.js)를 분리하여 개발했으며, 직관적인 UI와 캘린더 기반의 일정 관리 기능을 갖추고 있습니다.

---

## 🚀 기술 스택

- **Frontend**
  - React (Vite 기반)
  - FullCalendar
  - Axios
- **Backend**
  - Node.js (Express)
- **환경**
  - npm, Git, REST API
